# BlueBook

### 简介

一款社交类的软件，专注年轻人的笔记短视频社区，用户可以选择歌曲，配以短视频或图片，形成自己的作品，通过它你可以分享你的生活，同时也可以在这里认识到更多朋友，了解各种奇闻趣事。
